# 模式型技能定义 (Pattern Skill Definition)

## 概述 (Overview)

模式型技能（Pattern Skills）专注于思考问题的方式和解决方法的通用框架。关于五大模式的完整定义，请参阅 [skill-spec.md](skill-spec.md)。

## 与五大模式的关系 (Relationship with Five Modes)

模式型技能可以对应以下模式：

| 模式型技能类型 | 对应模式 | 说明 |
|---|---|---|
| 架构模式 | Pipeline | 多步骤流程串联 |
| 决策框架 | Inversion | 澄清后决策 |
| 审查模式 | Reviewer | 按清单核查 |

## 结构模板 (Structure Template)

```markdown
# 模式名称

## 概述 (Overview)

模式的核心思想和适用场景。

## 何时使用 (When to Use)

- 使用条件 1
- 使用条件 2

## 模式定义 (Pattern Definition)

模式的具体定义和组成部分。

## 应用步骤 (Application Steps)

1. 步骤 1
2. 步骤 2

## 示例 (Examples)

实际应用的示例。
```

## 常见错误 (Common Mistakes)

| 错误 | 解决方案 |
|---|---|
| 过于抽象 | 结合具体示例说明 |
| 不明确边界 | 清楚定义适用和不适用的场景 |
